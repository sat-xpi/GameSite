Phase 02 // Operations

三大核心模式

每种模式都有独特的战术逻辑，建议从烽火地带开始逐步体验。

🎯

烽火地带 — 搜打撤核心

3人小队战术生存玩法，类似《逃离塔科夫》。核心目标是搜刮资源并成功撤离。死亡将丢失本局所有物资（仅安全箱保留）。

推荐搜刮路线

西侧厂房 → 二楼保险箱（必出蓝紫色装备）→ 仓库区（医疗包、电子材料）→ 撤离

💰

地点产出

⚠

搜打撤执行要领

搜：优先级：黄标保险/骇客电脑 > 3级甲/头盔 > 武器弹药 > 医疗品 > 普通物资。一个区域不超1分钟，搜完立刻转点。

打：突发遇敌先丢烟雾弹 → 静步绕侧 → 瞄准头部连发 → 打完立刻换点。新手交火先蹲射（降后坐力、缩被击面积）。

撤：开局先记撤离点，标记2-3个备选。背包价值达20万左右、倒计时前5分钟出发。到达撤离点先清AI、排查埋伏。

军营潜入技巧

千万别走正门！用手雷炸通风口进入，安全系数翻倍。贵重品（大红、门禁卡、曼德尔砖）搜到立即存安全箱，阵亡不丢失。

⚔

全面战场 — 24v24大规模作战

可驾驶坦克、直升机等载具，适合有一定基础后体验大规模PVP。推荐职业：工程师。

  • 修复载具（装甲车、直升机），队友冲锋时蹲后方持续维修
  • 进攻方：优先抢占"棱镜攻势"航空箱获取额外优势
  • 防守方：B点失守后可绕后偷袭敌方A点，打乱进攻节奏
  • 坦克位：副炮位可快速刷分，被攻击时立即释放烟雾并维修
🛡

载具战术要点

  • 除快速移动外，载具可作为移动火力点提供支援
  • 工程师维修效率是其他干员的两倍
  • 直升机被锁定时立即趴下，空地也能规避锁定导弹
  • 团队协同：载具 + 步兵配合推进效率远高于单打独斗
建议

等级达到15级以上再进入全面战场体验大规模PVP。

🎥

黑鹰坠落 — PVE合作模式

还原经典电影战役，适合组队打副本拿奖励。任务示例：解救人质。

团队分工建议

  • 突击兵（红狼）：绕后清敌
  • 支援兵（蜂医）：烟雾掩护 + 治疗
  • 工程师：修复直升机
  • 侦察兵：标记敌方位置

关键技能保留

  • 导弹、群体烟雾等强力支援技能留到决胜时刻使用
  • 救援优先：队友倒地后优先救援，避免反复消耗复活资源
  • 禁止单独行动：组队分工效率更高
